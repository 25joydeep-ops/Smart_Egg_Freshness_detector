# Smart_Egg_Freshness_detector
An IoT-ML based solution to determine the freshness levels of an uncracked egg using a SVM-based ML model different sensors monitoring the egg and all computations performed using an ESP32 Wroom microcontroller.
The system consists of 3 sensors mounted insided a closed environment and measure the temperature & humidity of the area surrounding the egg, NH3 levels emitted by the egg and the air cell growth inside egg using optical candling. The final output is the classification of the egg's freshness into Fresh, Good, Marginal or Spoiled.
